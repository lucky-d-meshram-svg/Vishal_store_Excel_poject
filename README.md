  ## Project Overview
An Excel-based sales analysis dashboard built using Vishal Store retail data.  
**Get raw data:** [Click here](https://docs.google.com/spreadsheets/d/1IQielSXbz1Ae68brpTBwlXwQVU4IUUPD/edit?usp=drive_link&ouid=112301804410512564263&rtpof=true&sd=true)

  ## Key KPIs
  - Total Sales  
  - Total Orders  
  - Delivered / Cancelled / Returned Orders %  
  - Sales by Gender  
  - Sales by Age Group  
  - Top States by Sales  
  - Sales by Channel  
  
  ## KPI Logic
  - Total Sales: `SUM(Sales)`  
  - Total Orders: `COUNTA(Order ID)`  
  - Order %: `(Order Type / Total Orders) * 100`  
  - Other KPIs calculated using Pivot Tables
