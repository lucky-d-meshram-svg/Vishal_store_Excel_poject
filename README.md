#Project Overview

An Excel-based sales analysis dashboard built using Vishal Store retail data. The project analyzes sales performance using KPIs, pivot tables, and interactive visuals to generate business insights.

Key KPIs

Total Sales
Total Orders
Delivered / Cancelled / Returned Orders %
Sales by Gender and Age Group
Top States by Sales
Sales by Channel

KPI Logic

Total Sales: SUM(Sales)
Total Orders: COUNTA(Order ID)
Order %: (Order Type / Total Orders) * 100
Other KPIs calculated using Pivot Tables

Tools & Skills

Microsoft Excel
Data Cleaning
Pivot Tables & Charts
KPI Analysis
Dashboard Design

Business Value
This project demonstrates KPI-driven analysis and strong understanding of retail sales metrics, making it relevant for data analyst roles
